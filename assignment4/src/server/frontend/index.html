WickedDB School Database Client

Project description

Welcome to the 3309 final project deliverable!
To use this application, simply select the functionality you wish invoke by clicking one of the links below.
Each functionality has instructions on how to use the GUI to execute it.
The context of the app is that it is a database application that holds information regarding the entites involved in a school system.

Project Implementation Summary

The application was implemented using a very basic front-end structure with more complex back-end functionalities.
The front-end was built using basic html pages with minimal styling.
The back-end was built using an express server that facilitates page routing, as well as the database connection and queries.
The project also includes a number of helper node.js scripts that handle database initialization and data generation.

Functionalities

Functionality 1: Look up students by name and all related information
Functionality 2: Find the lowest salaried staff for all departments
Functionality 3: Given a student name and a course name, find other students in the same section
Functionality 4: Add a new student to the student relation
Functionality 5: Increase the salaries of all instructors of a department by a selected percent
Functionality 6: look up a student name and find all the courses they are taking

The quit functionality is present on all pages and serves
to terminate the current request by returning to the home page
